Welcome to math-atlas's git documentation!
==========================================

These pages provide help for accessing the basefiles ATLAS is developed in
using github_.  To actually understand what basefiles are, and how to
work with ATLAS source, you'll want to read the file 
*ATLAS/doc/atlas_devel.pdf* in your tarfile.  For general information,
see the `math-atlas home page`_, and the  `math-atlas developer home page`_.
